Tutorial Clase e Instanciacion

Crear la clase

Primero debemos crear la clase deseada para nuestro programa. Esta tiene que contener sus propiedades (privadas) y los metodos que serviran para acceder a ellas

Luego debemos importar esta nueva clase en la clase donde queremos utilizarla. Debemos recordar que si estan en el mismo paquete, este paso no es necesario

Procedemos a crear una instancia de este objeto, recordando brindarle los elementos necesarios para la Instanciacion

Y finalmente ya podemos utilizar la clase y modificar sus atributos a conveniencia
